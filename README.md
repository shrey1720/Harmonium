# Harmonium

A beautiful, keyboard-interactive Harmonium application built with React and Vite.

## 🎵 Features

- **Keyboard Interaction**: Play harmonium notes directly using your laptop keyboard.
- **Realistic Sound**: High-quality audio samples for an authentic harmonium experience.
- **Glassmorphic UI**: A modern, premium design with smooth animations and vibrant aesthetics.
- **Responsive Layout**: Works seamlessly across different screen sizes.

## 🚀 Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Modern CSS features)
- **Audio**: Web Audio API / HTML5 Audio

## 🛠️ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/shrey1720/Harmonium.git
    cd Harmonium
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
